This project shows how to use [OpenCV][cv1] C++ API to recognize features on
face such as `Age`, `Gender` and `Emotion` using [Histogram of Oriented
Gradients (HOG)][hog1] and [Support vector machine SVM][svm1] algorithms.

* [manid2/FacialFeaturesRecognizer][p1]

[p1]: https://github.com/manid2/FacialFeaturesRecognizer
[cv1]: https://opencv.org/
[hog1]: https://en.wikipedia.org/wiki/Histogram_of_oriented_gradients
[svm1]: https://en.wikipedia.org/wiki/Support_vector_machine
